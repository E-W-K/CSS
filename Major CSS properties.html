Difference between paddings, margins and borders

Padding is the space around an element's content, inside of any defined borders. Margin can be defined by left, right, top, bottom.
Margin is space around elements, outside of any defined borders.
Border is is a shorthand syntax in CSS that accepts multiple values for drawing a line around the element it is applied to.
